> ⚠️ **Cuidado:** esta sección modifica permisos del sistema. **No le saques permisos a tu propia cuenta** o te quedás afuera.

## Usuarios

Andá a **Acceso → Usuarios**.

- **Listar** usuarios. Filtrar por rol, estado.
- **Buscar** por email.
- Abrir un usuario al azar. Mirar las tabs (General, Actividad, Permisos).
- **Editar** un usuario (cambiar nombre, agregar permisos extra).
- **Crear** un usuario admin nuevo de prueba.

## Roles y permisos

> ⚠️ **NO toques las secciones "Roles" ni "Permisos"** (las listas maestras). Solo miralas. **No las edites.** Mirá [Lo que no tenés que tocar](/beta/admin-editor/no-tocar/).

## Qué reportar

> 📋 **Reportá:**
>
> - Si algo no guarda bien
> - Si un permiso aplicado no surge efecto
> - Si el listado tarda mucho en cargar
> - Si filtros o búsqueda no funcionan
