# Glosario rápido

Si una palabra te suena rara, mirá acá. Está dividido por temas.

## Lo básico de internet

| Palabra | Qué significa |
| --- | --- |
| **Dirección web (URL)** | La dirección del sitio, tipo `https://hospeda.com.ar`. También: link, enlace. |
| **Enlace (link)** | Algo donde hacés clic y te lleva a otro lado. |
| **Navegador** | El programa con el que abrís internet (Chrome, Firefox, Safari, Edge). |
| **Pestaña (tab)** | Cada "página" abierta dentro del navegador. |
| **Recargar** / "refrescar" | Volver a cargar la página. En compu: `F5` o `Ctrl + R` (`Cmd + R` en Mac). En celular: deslizar para abajo. |
| **Caché** | Lo que el navegador guarda para cargar más rápido. A veces hay que limpiarlo si algo se ve raro. |
| **Cookie** | Un dato chiquito que el sitio guarda en tu navegador (ej: para recordar tu sesión). |
| **Modo incógnito** / "privado" | Ventana que no guarda historial, cookies ni caché. Útil para probar como "usuario nuevo". |
| **HTTPS** | Que la dirección empiece con `https://` y tenga candadito. Conexión cifrada. |
| **404** | Error cuando un enlace te lleva a una página que no existe. |
| **500** | Error interno del servidor. No es tu culpa. **Reportalo.** |

## Lo que vas a ver en la pantalla

| Palabra | Qué significa |
| --- | --- |
| **Cabecera (header)** | La franja arriba de todo, con logo y menú principal. |
| **Pie de página (footer)** | La franja abajo de todo, con contacto, redes y enlaces secundarios. |
| **Barra de navegación (navbar)** | El menú con los enlaces principales (suele estar en la cabecera). |
| **Barra lateral (sidebar)** | Menú vertical al costado. Acá lo tenés a la izquierda. |
| **Ruta de navegación (breadcrumb)** | El rastro tipo "Inicio → Alojamientos → Casa de campo". Te muestra dónde estás. |
| **Botón** | Algo donde hacés clic para hacer una acción (texto, ícono, o ambos). |
| **CTA** | "Call to action" — el botón principal de una pantalla ("Suscribirme", "Reservar"). |
| **Enlace de texto (link)** | Palabra subrayada o de otro color dentro de un párrafo que actúa como enlace. |
| **Ícono** | El dibujito chiquito (corazón, lupa, engranaje). |
| **Ventana emergente (modal)** | Ventana flotante que aparece encima y oscurece el fondo. Se cierra con X o tocando afuera. |
| **Panel flotante (drawer)** | Ventana que entra deslizándose desde un costado. |
| **Menú desplegable (dropdown)** | El menú que se despliega al clickear una flechita o un campo. |
| **Notificación flotante (toast)** | Cartelito chiquito que aparece unos segundos para avisarte algo ("Guardado con éxito"). |
| **Aviso destacado (banner)** | Cartel ancho arriba con un aviso (ej: "Estás en modo beta"). |
| **Pop-up** | Cualquier ventana o cartel que aparece flotando sin que lo pidas. |
| **Texto de ejemplo (placeholder)** | Texto gris dentro de un campo vacío que te indica qué poner ("Tu email"). |
| **Ayuda emergente (tooltip)** | Cartelito que aparece cuando pasás el mouse encima de un ícono o botón. |
| **Pestaña (tab) en una página** | Solapas dentro de una página para cambiar de vista sin recargar. |
| **Esqueleto de carga (skeleton)** | Cajitas grises animadas que se ven mientras carga el contenido real. |
| **Indicador de carga (spinner)** | Círculo girando que indica "esperá, estoy cargando". |
| **Galería deslizable (carousel)** | Conjunto de fotos con flechitas o miniaturas para ir cambiando. |
| **Botón flotante (FAB)** | La pastilla "Reportar problema" abajo a la derecha (atajo `Ctrl + Shift + F`). |

## Acciones, formularios y cuentas

| Palabra | Qué significa |
| --- | --- |
| **Iniciar sesión (login)** | Entrar a tu cuenta con email y contraseña. |
| **Cerrar sesión (logout)** | Salir de tu cuenta. |
| **Crear cuenta (signup)** | Registrarte: dar de alta una cuenta nueva. |
| **Bienvenida inicial (onboarding)** | El primer recorrido que te muestra el sitio cuando te registrás. |
| **Sesión** | El tiempo que estás logueado. |
| **Formulario** | Conjunto de campos para escribir (nombre, email, etc.) más el botón de enviar. |
| **Campo** / "input" | Cada cuadradito donde escribís un dato. |
| **Validación** | Cuando el sistema chequea que lo que escribiste está bien (email con `@`, etc.). |
| **Error de validación** | Mensaje rojo abajo de un campo cuando algo está mal escrito. |
| **Enviar (submit)** | El botón que envía el formulario ("Guardar", "Registrarme"). |
| **Borrador (draft)** | Contenido guardado pero todavía no publicado. Lo ves vos, los demás no. |
| **Publicar** | Que el contenido pase de borrador a visible para todos. |
| **Email transaccional** | Email automático que te llega después de hacer algo (registrarte, pagar, resetear). |
| **Reset de contraseña** | El sistema te manda un enlace al email para crear una clave nueva. |

## Cosas específicas de Hospeda

| Palabra | Qué significa |
| --- | --- |
| **Propietario o anfitrión (host)** | Quien publica un alojamiento. |
| **Turista** / "guest" | El visitante que busca dónde alojarse. |
| **Lista (listing)** / "alojamiento" | Una unidad publicada (casa, departamento, cabaña). |
| **Detalle del alojamiento** | La página completa con galería, descripción, mapa, precio y contacto. |
| **Amenidad** | Servicio o comodidad del alojamiento (wifi, pileta, parrilla, aire). |
| **Atributo** | Características fijas (cantidad de camas, baños, m², capacidad). |
| **Destino** | Localidad o zona con alojamientos asociados. Tiene página propia con info turística. |
| **Evento** | Actividad turística con fecha (festival, deportivo, gastronómico). |
| **Post** / "publicación" | Un artículo del blog. |
| **Etiqueta (tag / chip / badge)** | Palabra clave que clasifica posts o eventos. |
| **Sponsor** | Auspiciante. Aparece destacado a cambio de pago. |
| **Plan de suscripción** | Membresía que paga el propietario para publicar (Free, Pro, Premium). |
| **Add-on** | Funcionalidad extra que se compra suelta encima de un plan. |
| **Código promo** | Cupón de descuento aplicable a la pantalla de pago. |
| **Modo de prueba (sandbox)** | Modo de prueba del sistema de pagos. **No usa plata real.** |
| **Pantalla de pago (checkout)** | La pantalla donde completás los datos para pagar. Cuando pagás un plan, corre en MercadoPago, no en Hospeda. |
| **MercadoPago** | El procesador de pagos que usamos. |
| **Tarjeta de prueba** | Tarjeta ficticia que solo funciona en modo de prueba. |
| **Panel de control (dashboard)** | Panel principal una vez logueado. Cambia según seas turista, propietario o admin. |
| **Panel admin** | Sitio aparte (`staging-admin.hospeda.com.ar`) para administradores y editores. |
| **Favorito** | Alojamiento que guardás para mirar después. |
| **Colección** | Grupo de favoritos que organizás vos (ej: "Viaje familia"). |

## Cosas del beta

| Palabra | Qué significa |
| --- | --- |
| **Beta** | Fase de prueba antes de abrir al público. |
| **Beta tester** | Vos. La persona que prueba el sitio antes del lanzamiento. |
| **Ambiente de prueba (staging)** | Entorno donde probamos. Copia separada del sitio real. La dirección arranca con `staging.`. |
| **Producción** / "prod" | El sitio real. **Durante el beta no estás tocando producción.** |
| **Error (bug)** | Algo que no funciona como esperás. |
| **Regresión** | Cuando algo que andaba bien se rompe después de un cambio. |
| **Reportar** | Avisar de un problema usando el botón flotante o el grupo de WhatsApp. |
| **Reproducir** | Volver a generar el mismo error a propósito, para demostrar que pasa. |
| **Pasos para reproducir** | Lista numerada de qué hiciste para que aparezca el error. |
| **Captura** / "screenshot" | Foto de lo que ves en tu pantalla. Mirá [Cómo sacar captura](#cómo-sacar-una-captura-de-pantalla). |
| **Grabación de pantalla** | Video corto. Útil cuando un error solo se ve "en movimiento". |
| **Tester preseleccionado** | Alguien avisado por privado para probar una sección específica (admin, pago real). |

## Cosas técnicas que aparecen en la docs

| Palabra | Qué significa |
| --- | --- |
| **Adaptable al dispositivo (responsive)** | Que el sitio se acomode bien a distintos tamaños de pantalla. |
| **Mobile first** | Diseño pensado primero para celular. |
| **i18n** | "Internacionalización". Que el sitio esté en varios idiomas. |
| **Locale** | Idioma activo. Hospeda tiene `es`, `en`, `pt`. |
| **Dark mode** | Variante visual con fondo oscuro. Hay un selector en la cabecera. |
| **Tema** | Variante visual completa (claro / oscuro / sistema). |
| **Console** / "consola" | Pantalla de mensajes técnicos del navegador (`F12`). Útil para mostrarnos errores rojos. |
| **DevTools** | El panel que se abre con `F12`: consola, network, elementos, etc. |
| **Sentry** | Servicio que captura errores automáticamente. No tenés que hacer nada con él. |
| **Código (token)** | Código alfanumérico que identifica una sesión o acceso. No los compartas. |

## Cómo sacar una captura de pantalla

| Dispositivo | Cómo |
| --- | --- |
| Celular Android | **Volumen abajo + Encendido** al mismo tiempo |
| iPhone con botón Home | **Home + Encendido** al mismo tiempo |
| iPhone sin Home | **Subir volumen + Encendido** al mismo tiempo |
| Windows | Tecla **PrtSc** o `Win + Shift + S` (selección) |
| Mac | `Cmd + Shift + 4` (selección) o `Cmd + Shift + 3` (pantalla completa) |
| Linux | `PrtSc` o usá la app "Captura de pantalla" |

> Si necesitás grabar video de la pantalla, mirá **["Herramientas para testear"](/beta/faq/#herramientas-para-testear)** en las FAQ.
