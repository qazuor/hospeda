# Glosario rápido

Si una palabra te suena rara, mirá acá primero. Está dividido por temas para que encuentres rápido.

## Lo básico de internet

| Palabra | Qué significa |
| --- | --- |
| **URL** | La dirección del sitio. Tipo `https://hospeda.com.ar`. Sinónimos: link, dirección, enlace. |
| **Link** | Lo mismo que URL. Cualquier cosa donde clickeás y te lleva a otro lado. |
| **Navegador** | El programa con el que abrís internet (Chrome, Firefox, Safari, Edge). |
| **Pestaña** o "tab" | Cada "página" abierta dentro del navegador. Las podés tener varias al mismo tiempo. |
| **Recargar** o "refrescar" | Volver a cargar la página actual. En compu: tecla `F5` o `Ctrl + R` (`Cmd + R` en Mac). En celular: deslizar para abajo. |
| **Caché** | Lo que el navegador guarda en tu equipo para que la próxima vez cargue más rápido. A veces hay que limpiarlo si algo se ve raro. |
| **Cookie** | Un dato chiquito que el sitio te guarda en el navegador (por ejemplo, para recordar que iniciaste sesión). |
| **Modo incógnito** o "privado" | Una ventana del navegador que no guarda historial, cookies ni caché. Útil para probar como "usuario nuevo". |
| **HTTPS** | Que la URL empiece con `https://` y tenga el candadito. Significa que la conexión está cifrada. |
| **404** | El error que te aparece cuando un link te lleva a una página que no existe. |
| **500** | Un error interno del servidor. No es tu culpa. **Reportalo.** |

## Lo que vas a ver en la pantalla

| Palabra | Qué significa |
| --- | --- |
| **Header** | La franja arriba de todo, con el logo y el menú principal. |
| **Footer** | La franja abajo de todo, con los datos de contacto, redes y links secundarios. |
| **Navbar** o "barra de navegación" | El menú con los links principales del sitio (suele estar en el header). |
| **Sidebar** o "barra lateral" | El menú vertical al costado. Acá en el docs site lo tenés a la izquierda. |
| **Breadcrumb** | El rastro de migas: "Inicio → Alojamientos → Casa de campo". Te muestra dónde estás parado en el sitio. |
| **Botón** | Algo donde clickeás para hacer una acción. Puede ser texto, ícono, o ambos. |
| **CTA** | "Call to action" — el botón principal de una pantalla ("Suscribirme", "Empezar", "Reservar"). |
| **Link de texto** | Una palabra subrayada o de otro color dentro de un párrafo que actúa como link. |
| **Ícono** | El dibujito chiquito (corazón, lupa, engranaje, etc.). |
| **Modal** o "diálogo" | La ventana flotante que aparece encima de la pantalla y oscurece el fondo. La cerrás con la X o tocando afuera. |
| **Drawer** o "panel lateral" | Una ventana que entra deslizándose desde un costado (no flota como el modal). |
| **Dropdown** | El menú que se despliega cuando clickeás en una flechita o un campo (lista de opciones). |
| **Toast** o "snackbar" | El cartelito chiquito que aparece arriba o abajo unos segundos para avisarte algo ("Guardado con éxito"). |
| **Banner** | Un cartel ancho en la parte de arriba con un aviso (ej: "Estás en modo beta"). |
| **Pop-up** | Cualquier ventana o cartel que aparece flotando sin que lo pidas. |
| **Placeholder** | El texto gris dentro de un campo vacío que te indica qué poner ahí ("Tu email"). Desaparece cuando empezás a escribir. |
| **Tooltip** | El cartelito que aparece cuando pasás el mouse encima de un ícono o botón. |
| **Tab** o "pestaña" (en una página) | Las solapas dentro de una página para cambiar de vista sin recargar. |
| **Skeleton** | Las cajitas grises animadas que se ven mientras carga el contenido real. |
| **Spinner** o "loader" | El círculo girando que indica "esperá, estoy cargando". |
| **Galería** | El conjunto de fotos con flechitas o miniaturas para ir cambiando. |
| **FAB** o "botón flotante" | La pastilla flotante "Reportar problema" abajo a la derecha (atajo `Ctrl + Shift + F`). |

## Acciones, formularios y cuentas

| Palabra | Qué significa |
| --- | --- |
| **Login** / "iniciar sesión" | Entrar a tu cuenta con email y contraseña. |
| **Logout** / "cerrar sesión" | Salir de tu cuenta. |
| **Signup** / "registrarse" | Crear una cuenta nueva. |
| **Onboarding** | El primer recorrido que te muestra el sitio cuando te registrás. |
| **Sesión** | El tiempo que estás logueado. Si cierra, hay que loguearse de nuevo. |
| **Formulario** | El conjunto de campos para escribir (nombre, email, etc.) más el botón de enviar. |
| **Campo** o "input" | Cada cuadradito donde escribís un dato (nombre, email, teléfono). |
| **Validación** | Cuando el sistema chequea que lo que escribiste está bien (email con `@`, contraseña larga, etc.). |
| **Error de validación** | El mensaje rojo abajo de un campo cuando algo está mal escrito. |
| **Submit** | El botón que envía el formulario ("Guardar", "Enviar", "Registrarme"). |
| **Draft** o "borrador" | Un contenido guardado pero todavía no publicado. Vos lo ves, los demás no. |
| **Publicar** | Que el contenido pase de borrador a estado público / visible para todos. |
| **Email transaccional** | El email automático que te llega después de hacer algo (registrarte, pagar, resetear contraseña, etc.). |
| **Reset de contraseña** | Cuando te olvidaste la clave: el sistema te manda un link al email para crear una nueva. |

## Cosas específicas de Hospeda

| Palabra | Qué significa |
| --- | --- |
| **Host** | Propietario que publica un alojamiento. |
| **Turista** o "guest" | El visitante que busca dónde alojarse. |
| **Listing** o "alojamiento" | Una unidad publicada (casa, departamento, cabaña, etc.). |
| **Detalle del alojamiento** | La página completa de un alojamiento, con galería, descripción, mapa, precio y formulario de contacto. |
| **Amenidad** o "amenity" | Cada servicio o comodidad del alojamiento (wifi, pileta, parrilla, aire, etc.). |
| **Atributo** | Características fijas del alojamiento (cantidad de camas, baños, m², capacidad). |
| **Destino** | Una localidad o zona que tiene alojamientos asociados. Tiene su propia página con info turística. |
| **Evento** | Una actividad turística con fecha (festival, fiesta, deportivo, gastronómico). |
| **Post** o "publicación" | Un artículo del blog. |
| **Tag** o "etiqueta" | Palabra clave que clasifica posts o eventos. |
| **Sponsor** | Auspiciante. Aparece destacado en alguna sección del sitio a cambio de pago. |
| **Plan de suscripción** | La membresía que paga el host para publicar (Free, Pro, Premium, etc.). |
| **Add-on** | Una funcionalidad extra que se compra suelta encima de un plan (más fotos, destacar listing, etc.). |
| **Código promo** | Cupón de descuento aplicable al checkout. |
| **Sandbox** | Modo de prueba del sistema de pagos. **No usa plata real.** |
| **Checkout** | La pantalla donde completás los datos para pagar. |
| **MercadoPago** | El procesador de pagos que usamos. |
| **Tarjeta de prueba** | Una tarjeta ficticia que solo funciona en sandbox. Te las pasamos en la documentación. |
| **Dashboard** | El panel principal una vez que iniciás sesión. Cambia según seas turista, host o admin. |
| **Panel admin** | El sitio aparte (`staging-admin.hospeda.com.ar`) donde administradores y editores gestionan el contenido. |
| **Favorito** | Un alojamiento que guardás para mirar después. Aparecen en tu cuenta. |
| **Colección** | Un grupo de favoritos que vos organizás (ej: "Viaje familia", "Escapadas en pareja"). |

## Cosas del beta

| Palabra | Qué significa |
| --- | --- |
| **Beta** | Una fase de prueba antes de abrir al público. |
| **Beta tester** | Vos. La persona que prueba el sitio antes del lanzamiento. |
| **Staging** | El entorno donde probamos. Es una copia separada del sitio real. La URL siempre arranca con `staging.`. |
| **Producción** o "prod" | El sitio real, el que va a ver el público. **Durante el beta no estás tocando producción.** |
| **Bug** | Un error: algo que no funciona como esperás. |
| **Regresión** | Cuando algo que andaba bien se rompe después de un cambio. |
| **Reportar** | Avisar de un bug o problema usando el FAB o el grupo de WhatsApp. |
| **Reproducir** | Volver a generar el mismo error a propósito, para demostrar que pasa. |
| **Pasos para reproducir** | La lista numerada de qué hiciste para que aparezca el bug ("1. Entré a X, 2. Hice click en Y, 3. Apareció error"). |
| **Captura** o "screenshot" | Una foto de lo que ves en tu pantalla. Mirá [Cómo sacar captura](#cómo-sacar-una-captura-de-pantalla). |
| **Grabación de pantalla** | Un video corto que muestra qué hacés mientras se ve la pantalla. Útil cuando un bug solo se ve "en movimiento". |
| **Tester preseleccionado** | Alguien a quien le avisamos por privado que iba a probar una sección específica (admin, pago real). |

## Cosas técnicas que aparecen en la docs

| Palabra | Qué significa |
| --- | --- |
| **Responsive** | Que el sitio se acomode bien a distintos tamaños de pantalla (celular, tablet, compu). |
| **Mobile first** | Diseño pensado primero para celular y después adaptado a pantallas más grandes. |
| **i18n** | "Internacionalización". Que el sitio esté disponible en varios idiomas. |
| **Locale** o "idioma" | El idioma activo. Hospeda tiene `es` (español), `en` (inglés), `pt` (portugués). |
| **Dark mode** o "modo oscuro" | Variante visual con fondo oscuro. Hay un selector en el header. |
| **Tema** | Variante visual completa (claro / oscuro / sistema). |
| **Console** o "consola" | Una pantalla de mensajes técnicos del navegador. Se abre con `F12`. Útil para mostrarnos errores rojos. |
| **DevTools** o "herramientas de desarrollador" | El panel que se abre con `F12` y que tiene la consola, network, elementos, etc. |
| **Sentry** | Un servicio que captura errores automáticamente del sitio. No tenés que hacer nada con él. |
| **Token** | Un código alfanumérico que identifica una sesión o un acceso. No los compartas. |

## Cómo sacar una captura de pantalla

| Dispositivo | Cómo |
| --- | --- |
| Celular Android | **Volumen abajo + Encendido** al mismo tiempo |
| iPhone con botón Home | **Home + Encendido** al mismo tiempo |
| iPhone sin Home | **Subir volumen + Encendido** al mismo tiempo |
| Windows | Tecla **PrtSc** (imprimir pantalla) o `Win + Shift + S` (selección) |
| Mac | `Cmd + Shift + 4` (selección) o `Cmd + Shift + 3` (pantalla completa) |
| Linux | `PrtSc` o usá la app "Captura de pantalla" |

> Si necesitás grabar un video corto de la pantalla (algunos bugs solo se ven en movimiento), mirá la sección **["Herramientas para testear"](/beta/faq/#herramientas-para-testear)** en las FAQ.
