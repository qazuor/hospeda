# Glosario rápido

Por si hace falta. Si una palabra te suena rara, mirá acá primero.

| Palabra | Qué significa |
| --- | --- |
| **URL** | La dirección del sitio web. Tipo `https://hospeda.com.ar` |
| **Link** | Lo mismo que URL. Algo donde clickeás y te lleva a otro lado. |
| **Bug** | Un error: algo que no funciona como esperás |
| **FAB** o "botón flotante" | La pastilla flotante "Reportar problema" abajo a la derecha (atajo `Ctrl + Shift + F`) |
| **Captura de pantalla** o "screenshot" | Una foto de lo que ves en tu pantalla |
| **Login** | Iniciar sesión, entrar a tu cuenta |
| **Logout** | Cerrar sesión, salir de tu cuenta |
| **Sandbox** | Modo de prueba (no usa plata real) |
| **Host** | Propietario que publica un alojamiento |
| **Listing** | Un alojamiento publicado |
| **Checkout** | La pantalla donde completás los datos para pagar |
| **CTA** | "Call to action" — botón principal de una pantalla ("Empezar", "Suscribirme") |
| **Onboarding** | El primer recorrido de un usuario nuevo en la plataforma |
| **Dashboard** | Panel de control. La pantalla principal cuando entrás logueado. |
| **i18n** | Internacionalización (que el sitio esté en varios idiomas) |
| **Responsive** | Que el sitio se acomode bien a distintos tamaños de pantalla |
| **Email transaccional** | El email que te llega después de hacer algo (registrarte, pagar, etc.) |

## Cómo sacar una captura de pantalla

| Dispositivo | Cómo |
| --- | --- |
| Celular Android | **Volumen abajo + Encendido** al mismo tiempo |
| iPhone con botón Home | **Home + Encendido** al mismo tiempo |
| iPhone sin Home | **Subir volumen + Encendido** al mismo tiempo |
| Windows | Tecla **PrtSc** (imprimir pantalla) o `Win + Shift + S` (selección) |
| Mac | `Cmd + Shift + 4` (selección) o `Cmd + Shift + 3` (pantalla completa) |
| Linux | `PrtSc` o usá la app "Captura de pantalla" |
