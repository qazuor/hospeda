# Glosario rápido

Si una palabra te suena rara, mirá acá primero. Está dividido por temas para que encuentres rápido.

## Lo básico de internet

| Palabra | Qué significa |
| --- | --- |
| **Dirección web (URL)** | La dirección del sitio. Tipo `https://hospeda.com.ar`. También se le dice **URL**, link, enlace. |
| **Enlace (link)** | Lo mismo que una dirección web. Cualquier cosa donde clickeás y te lleva a otro lado. También se le dice **link**. |
| **Navegador** | El programa con el que abrís internet (Chrome, Firefox, Safari, Edge). |
| **Pestaña (tab)** | Cada "página" abierta dentro del navegador. Las podés tener varias al mismo tiempo. También se le dice **tab**. |
| **Recargar** o "refrescar" | Volver a cargar la página actual. En computadora: tecla `F5` o `Ctrl + R` (`Cmd + R` en Mac). En celular: deslizar para abajo. |
| **Caché** | Lo que el navegador guarda en tu equipo para que la próxima vez cargue más rápido. A veces hay que limpiarlo si algo se ve raro. |
| **Cookie** | Un dato chiquito que el sitio te guarda en el navegador (por ejemplo, para recordar que iniciaste sesión). |
| **Modo incógnito** o "privado" | Una ventana del navegador que no guarda historial, cookies ni caché. Útil para probar como "usuario nuevo". |
| **HTTPS** | Que la dirección web empiece con `https://` y tenga el candadito. Significa que la conexión está cifrada. |
| **404** | El error que te aparece cuando un enlace te lleva a una página que no existe. |
| **500** | Un error interno del servidor. No es tu culpa. **Reportalo.** |

## Lo que vas a ver en la pantalla

| Palabra | Qué significa |
| --- | --- |
| **Cabecera (header)** | La franja arriba de todo, con el logo y el menú principal. También se le dice **header**. |
| **Pie de página (footer)** | La franja abajo de todo, con los datos de contacto, redes y enlaces secundarios. También se le dice **footer**. |
| **Barra de navegación (navbar)** | El menú con los enlaces principales del sitio (suele estar en la cabecera). También se le dice **navbar**. |
| **Barra lateral (sidebar)** | El menú vertical al costado. Acá en el docs site lo tenés a la izquierda. También se le dice **sidebar**. |
| **Ruta de navegación (breadcrumb)** | El rastro de migas: "Inicio → Alojamientos → Casa de campo". Te muestra dónde estás parado en el sitio. También se le dice **breadcrumb**. |
| **Botón** | Algo donde clickeás para hacer una acción. Puede ser texto, ícono, o ambos. |
| **CTA** | "Call to action" — el botón principal de una pantalla ("Suscribirme", "Empezar", "Reservar"). |
| **Enlace de texto (link)** | Una palabra subrayada o de otro color dentro de un párrafo que actúa como enlace. También se le dice **link**. |
| **Ícono** | El dibujito chiquito (corazón, lupa, engranaje, etc.). |
| **Ventana emergente (modal)** | La ventana flotante que aparece encima de la pantalla y oscurece el fondo. La cerrás con la X o tocando afuera. También se le dice **modal** o "diálogo". |
| **Panel flotante (drawer)** | Una ventana que entra deslizándose desde un costado (no flota como una ventana emergente). También se le dice **drawer** o "panel lateral". |
| **Menú desplegable (dropdown)** | El menú que se despliega cuando clickeás en una flechita o un campo (lista de opciones). También se le dice **dropdown**. |
| **Notificación flotante (toast / snackbar)** | El cartelito chiquito que aparece arriba o abajo unos segundos para avisarte algo ("Guardado con éxito"). También se le dice **toast** o **snackbar**. |
| **Aviso destacado (banner)** | Un cartel ancho en la parte de arriba con un aviso (ej: "Estás en modo beta"). También se le dice **banner**. |
| **Pop-up** | Cualquier ventana o cartel que aparece flotando sin que lo pidas. |
| **Texto de ejemplo (placeholder)** | El texto gris dentro de un campo vacío que te indica qué poner ahí ("Tu email"). Desaparece cuando empezás a escribir. También se le dice **placeholder**. |
| **Ayuda emergente (tooltip)** | El cartelito que aparece cuando pasás el mouse encima de un ícono o botón. También se le dice **tooltip**. |
| **Pestaña (tab)** (en una página) | Las solapas dentro de una página para cambiar de vista sin recargar. También se le dice **tab**. |
| **Esqueleto de carga (skeleton)** | Las cajitas grises animadas que se ven mientras carga el contenido real. También se le dice **skeleton**. |
| **Indicador de carga (spinner / loader)** | El círculo girando que indica "esperá, estoy cargando". También se le dice **spinner** o **loader**. |
| **Galería deslizable (carousel)** | El conjunto de fotos con flechitas o miniaturas para ir cambiando. También se le dice **carousel** o **slider** (cuando es de imágenes). |
| **Botón flotante (FAB)** | La pastilla flotante "Reportar problema" abajo a la derecha (atajo `Ctrl + Shift + F`). También se le dice **FAB** (floating action button). |

## Acciones, formularios y cuentas

| Palabra | Qué significa |
| --- | --- |
| **Iniciar sesión (login)** | Entrar a tu cuenta con email y contraseña. También se le dice **login**. |
| **Cerrar sesión (logout)** | Salir de tu cuenta. También se le dice **logout**. |
| **Crear cuenta (signup)** | Registrarse: dar de alta una cuenta nueva. También se le dice **signup**. |
| **Bienvenida inicial (onboarding)** | El primer recorrido que te muestra el sitio cuando te registrás. También se le dice **onboarding**. |
| **Sesión** | El tiempo que estás logueado. Si cierra, hay que iniciar sesión de nuevo. |
| **Formulario** | El conjunto de campos para escribir (nombre, email, etc.) más el botón de enviar. |
| **Campo** o "input" | Cada cuadradito donde escribís un dato (nombre, email, teléfono). |
| **Validación** | Cuando el sistema chequea que lo que escribiste está bien (email con `@`, contraseña larga, etc.). |
| **Error de validación** | El mensaje rojo abajo de un campo cuando algo está mal escrito. |
| **Enviar (submit)** | El botón que envía el formulario ("Guardar", "Enviar", "Registrarme"). También se le dice **submit**. |
| **Borrador (draft)** | Un contenido guardado pero todavía no publicado. Vos lo ves, los demás no. También se le dice **draft**. |
| **Publicar** | Que el contenido pase de borrador a estado público / visible para todos. |
| **Email transaccional** | El email automático que te llega después de hacer algo (registrarte, pagar, resetear contraseña, etc.). |
| **Reset de contraseña** | Cuando te olvidaste la clave: el sistema te manda un enlace al email para crear una nueva. |

## Cosas específicas de Hospeda

| Palabra | Qué significa |
| --- | --- |
| **Propietario o anfitrión (host)** | Propietario que publica un alojamiento. También se le dice **host** o anfitrión. |
| **Turista** o "guest" | El visitante que busca dónde alojarse. |
| **Lista (listing)** o "alojamiento" | Una unidad publicada (casa, departamento, cabaña, etc.). También se le dice **listing**. |
| **Detalle del alojamiento** | La página completa de un alojamiento, con galería, descripción, mapa, precio y formulario de contacto. |
| **Amenidad** o "amenity" | Cada servicio o comodidad del alojamiento (wifi, pileta, parrilla, aire, etc.). |
| **Atributo** | Características fijas del alojamiento (cantidad de camas, baños, m², capacidad). |
| **Destino** | Una localidad o zona que tiene alojamientos asociados. Tiene su propia página con info turística. |
| **Evento** | Una actividad turística con fecha (festival, fiesta, deportivo, gastronómico). |
| **Post** o "publicación" | Un artículo del blog. |
| **Etiqueta (tag / chip / badge)** | Palabra clave que clasifica posts o eventos. También se le dice **tag**, **chip** o **badge**. |
| **Sponsor** | Auspiciante. Aparece destacado en alguna sección del sitio a cambio de pago. |
| **Plan de suscripción** | La membresía que paga el propietario para publicar (Free, Pro, Premium, etc.). |
| **Add-on** | Una funcionalidad extra que se compra suelta encima de un plan (más fotos, destacar lista, etc.). |
| **Código promo** | Cupón de descuento aplicable al checkout. |
| **Modo de prueba (sandbox)** | Modo de prueba del sistema de pagos. **No usa plata real.** También se le dice **sandbox**. |
| **Checkout** | La pantalla donde completás los datos para pagar. |
| **MercadoPago** | El procesador de pagos que usamos. |
| **Tarjeta de prueba** | Una tarjeta ficticia que solo funciona en modo de prueba. Te las pasamos en la documentación. |
| **Panel de control (dashboard)** | El panel principal una vez que iniciás sesión. Cambia según seas turista, propietario o admin. También se le dice **dashboard**. |
| **Panel admin** | El sitio aparte (`staging-admin.hospeda.com.ar`) donde administradores y editores gestionan el contenido. |
| **Favorito** | Un alojamiento que guardás para mirar después. Aparecen en tu cuenta. |
| **Colección** | Un grupo de favoritos que vos organizás (ej: "Viaje familia", "Escapadas en pareja"). |

## Cosas del beta

| Palabra | Qué significa |
| --- | --- |
| **Beta** | Una fase de prueba antes de abrir al público. |
| **Beta tester** | Vos. La persona que prueba el sitio antes del lanzamiento. |
| **Ambiente de prueba (staging)** | El entorno donde probamos. Es una copia separada del sitio real. La dirección web siempre arranca con `staging.`. También se le dice **staging**. |
| **Producción** o "prod" | El sitio real, el que va a ver el público. **Durante el beta no estás tocando producción.** |
| **Error (bug)** | Algo que no funciona como esperás. También se le dice **bug**. |
| **Regresión** | Cuando algo que andaba bien se rompe después de un cambio. |
| **Reportar** | Avisar de un error o problema usando el botón flotante o el grupo de WhatsApp. |
| **Reproducir** | Volver a generar el mismo error a propósito, para demostrar que pasa. |
| **Pasos para reproducir** | La lista numerada de qué hiciste para que aparezca el error ("1. Entré a X, 2. Hice click en Y, 3. Apareció error"). |
| **Captura** o "screenshot" | Una foto de lo que ves en tu pantalla. Mirá [Cómo sacar captura](#cómo-sacar-una-captura-de-pantalla). |
| **Grabación de pantalla** | Un video corto que muestra qué hacés mientras se ve la pantalla. Útil cuando un error solo se ve "en movimiento". |
| **Tester preseleccionado** | Alguien a quien le avisamos por privado que iba a probar una sección específica (admin, pago real). |

## Cosas técnicas que aparecen en la docs

| Palabra | Qué significa |
| --- | --- |
| **Adaptable al dispositivo (responsive)** | Que el sitio se acomode bien a distintos tamaños de pantalla (celular, tablet, computadora). También se le dice **responsive**. |
| **Mobile first** | Diseño pensado primero para celular y después adaptado a pantallas más grandes. |
| **i18n** | "Internacionalización". Que el sitio esté disponible en varios idiomas. |
| **Locale** o "idioma" | El idioma activo. Hospeda tiene `es` (español), `en` (inglés), `pt` (portugués). |
| **Dark mode** o "modo oscuro" | Variante visual con fondo oscuro. Hay un selector en la cabecera. |
| **Tema** | Variante visual completa (claro / oscuro / sistema). |
| **Console** o "consola" | Una pantalla de mensajes técnicos del navegador. Se abre con `F12`. Útil para mostrarnos errores rojos. |
| **DevTools** o "herramientas de desarrollador" | El panel que se abre con `F12` y que tiene la consola, network, elementos, etc. |
| **Sentry** | Un servicio que captura errores automáticamente del sitio. No tenés que hacer nada con él. |
| **Código (token)** | Un código alfanumérico que identifica una sesión o un acceso. No los compartas. También se le dice **token**. |

## Cómo sacar una captura de pantalla

| Dispositivo | Cómo |
| --- | --- |
| Celular Android | **Volumen abajo + Encendido** al mismo tiempo |
| iPhone con botón Home | **Home + Encendido** al mismo tiempo |
| iPhone sin Home | **Subir volumen + Encendido** al mismo tiempo |
| Windows | Tecla **PrtSc** (imprimir pantalla) o `Win + Shift + S` (selección) |
| Mac | `Cmd + Shift + 4` (selección) o `Cmd + Shift + 3` (pantalla completa) |
| Linux | `PrtSc` o usá la app "Captura de pantalla" |

> Si necesitás grabar un video corto de la pantalla (algunos errores solo se ven en movimiento), mirá la sección **["Herramientas para testear"](/beta/faq/#herramientas-para-testear)** en las FAQ.
