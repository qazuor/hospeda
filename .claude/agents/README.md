# Agents

Specialized agent prompts for Claude Code delegation.

## Categories

- `product/` - Product management agents (functional specs, requirements)
- `engineering/` - Engineering agents (framework specialists, tech leads)
- `quality/` - Quality assurance agents (testing, debugging)
- `design/` - Design agents (UX/UI, content writing)
- `specialized/` - Domain-specific agents (i18n, SEO, documentation)

## Usage

Agents are referenced by the coordinating Claude Code instance via the Task tool. Each agent file contains the system prompt and instructions for that specialized role.
